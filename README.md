# Phishing Website Detection using Machine Learning

## Objective

Develop a machine learning model capable of classifying phishing and legitimate websites using the PhiUSIIL dataset.

---

## Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
- LightGBM
- CatBoost
- SHAP
- Streamlit (later)

---

## Dataset

PhiUSIIL Phishing Website Dataset (UCI Repository)
